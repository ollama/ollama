Goose is Block's open-source coding agent. Ollama can launch Goose Desktop or Goose CLI with Ollama configured as the model provider.

## Install

Install [Goose](https://goose-docs.ai/docs/getting-started/installation/).

<Note>Ollama can launch Goose Desktop on macOS and Windows. On Linux, use Goose CLI with `ollama launch goose-cli`.</Note>

## Quick setup

### Goose Desktop

```shell
ollama launch goose
```

`goose` is an alias for `goose-desktop`, so this command is equivalent:

```shell
ollama launch goose-desktop
```

Ollama selects a model, configures Goose to use the `ollama` provider, and opens Goose Desktop.

To launch Goose Desktop with a specific model:

```shell
ollama launch goose --model qwen3
```

If Goose Desktop is already running, Ollama asks before restarting it so the selected model takes effect.

### Goose CLI

```shell
ollama launch goose-cli
```

Ollama starts `goose session` with the `ollama` provider configured for the selected model.

To launch Goose CLI with a specific model:

```shell
ollama launch goose-cli --model qwen3
```

To pass arguments to Goose CLI, put them after `--`:

```shell
ollama launch goose-cli -- --debug
```

## Ollama Cloud

Use an Ollama Cloud model by passing the model name:

```shell
ollama launch goose --model qwen3-coder:480b-cloud
```

For Goose CLI:

```shell
ollama launch goose-cli --model qwen3-coder:480b-cloud
```

To connect Goose directly to ollama.com, set `OLLAMA_HOST` and `OLLAMA_API_KEY` before launching:

```shell
OLLAMA_HOST=https://ollama.com OLLAMA_API_KEY=your-api-key ollama launch goose --model qwen3-coder:480b-cloud
```

## Manual setup

If you prefer to configure Goose yourself, choose the Ollama provider in Goose and use:

```text
http://localhost:11434
```

For Goose CLI, run:

```shell
goose configure
```

Select **Configure Providers**, choose **Ollama**, and enter the model you want to use.
