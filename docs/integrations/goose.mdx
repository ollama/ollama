Goose is Block's open-source coding agent for desktop and terminal workflows. Ollama configures Goose to use Ollama as the model provider, so Goose can work with local models and Ollama Cloud models.

## Install

Install [Goose Desktop or Goose CLI](https://goose-docs.ai/docs/getting-started/installation/).

<Note>Ollama can launch Goose Desktop on macOS and Windows. On Linux, use Goose CLI with `ollama launch goose-cli`.</Note>

## Usage with Ollama

### Goose Desktop

```shell
ollama launch goose
```

`goose` is an alias for `goose-desktop`, so this command is equivalent:

```shell
ollama launch goose-desktop
```

When launched through `ollama launch goose`, Ollama selects a model, configures Goose to use the `ollama` provider, and opens Goose Desktop.

### Run Goose Desktop directly with a model

```shell
ollama launch goose --model qwen3
```

If Goose Desktop is already running, Ollama asks before restarting it so the selected model takes effect.

### Goose CLI

```shell
ollama launch goose-cli
```

Ollama starts `goose session` with the `ollama` provider configured for the selected model.

### Run Goose CLI directly with a model

```shell
ollama launch goose-cli --model qwen3
```

To pass arguments to Goose CLI, put them after `--`:

```shell
ollama launch goose-cli -- --debug
```

## Ollama Cloud

To use a cloud model:

```shell
ollama launch goose --model kimi-k2.7-code:cloud
```

For Goose CLI:

```shell
ollama launch goose-cli --model kimi-k2.7-code:cloud
```

To connect Goose CLI directly to ollama.com, create an [API key](https://ollama.com/settings/keys), then set `OLLAMA_HOST` and `OLLAMA_API_KEY` before launching:

```shell
OLLAMA_HOST=https://ollama.com OLLAMA_API_KEY=your-api-key ollama launch goose-cli --model kimi-k2.7-code:cloud
```

## Manual setup

If you prefer to configure Goose yourself, choose the Ollama provider in Goose and use:

```text
http://localhost:11434
```

For Goose CLI, run:

```shell
goose configure
```

Select **Configure Providers**, choose **Ollama**, and enter the model you want to use.
