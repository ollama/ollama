VS Code includes built-in AI chat through GitHub Copilot Chat. Use Ollama models directly in the Copilot Chat model picker.

## Get started

Launch VS Code with a model:

```shell
ollama launch vscode --model gemma4:cloud
```

Ollama installs the Ollama VS Code extension if needed, opens VS Code, and preselects the model in the Copilot Chat model picker.

## Manual setup

You'll need:

- [Visual Studio Code 1.120 or newer](https://code.visualstudio.com/download)
- Ollama installed and running
- At least one local or cloud model available in Ollama

<p className="manual-step-title">1. Install the Ollama extension</p>

Install the [Ollama extension](https://marketplace.visualstudio.com/items?itemName=Ollama.ollama) from the VS Code Marketplace.

<p className="manual-step-title">2. Start Ollama</p>

Make sure Ollama is running and at least one local or cloud model is available.

<p className="manual-step-title">3. Open VS Code Chat</p>

Open Chat in VS Code.

<p className="manual-step-title">4. Choose an Ollama model</p>

Open the model picker at the bottom of the chat input, then choose a model from the **Ollama** section.

The extension discovers models from `http://127.0.0.1:11434` by default.

<Note>
If models do not appear, run **Ollama: Refresh Models** from the Command Palette. Use **Ollama: Diagnose Models** to troubleshoot model detection in the Ollama output channel.
</Note>
