---
title: Hermes Agent
---

Hermes Agent is an assistant from Nous Research.

## Quick start

```bash
ollama launch hermes
```

This installs Hermes if needed, writes the Ollama-backed config to `~/.hermes/config.yaml`, and hands off to Hermes for gateway setup on first run.

## Manual install

```bash
curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash
```

To configure without launching chat:

```bash
ollama launch hermes --config
```
