[OpenClaw](https://openclaw.ai) is a personal AI assistant that runs on your own devices. It bridges messaging services (WhatsApp, Telegram, Slack, Discord, iMessage, and more) to AI coding agents through a centralized gateway.

## Get started

Launch OpenClaw with Ollama:

```shell
ollama launch openclaw
```

If OpenClaw is not installed, Ollama prompts to install it with npm.

On first launch, Ollama shows a security notice, configures OpenClaw with your selected model, starts the gateway, and launches OpenClaw.

## Models

Choose a model with enough context for the assistant and its tools.

<CardGroup cols={2}>
  <Card title="Cloud models" icon="cloud" href="https://ollama.com/search?c=cloud">
    Use larger models without downloading them.
  </Card>

  <Card title="Models with long context" icon="hard-drive" href="https://ollama.com/search?c=thinking">
    Browse models with large context windows.
  </Card>
</CardGroup>

Run OpenClaw directly with a model:

```shell
ollama launch openclaw --model gemma4:cloud
```

<Note>For local models, set the [context length](/context-length) to 64k or higher.</Note>

## Channels

On first interactive launch, Ollama asks whether you want to connect a messaging channel.

To connect one later:

```shell
openclaw channels add
```

OpenClaw supports channels such as WhatsApp, Telegram, Discord, Slack, and iMessage.

## Configure without launching

Update the OpenClaw model configuration without starting the gateway and launching OpenClaw:

```shell
ollama launch openclaw --config
```

Run OpenClaw without interaction for scripts or CI:

```shell
ollama launch openclaw --model gemma4:cloud --yes
```

## Manual setup

<p className="manual-step-title">1. Install OpenClaw</p>

```shell
npm install -g openclaw@latest
```

<p className="manual-step-title">2. Run onboarding</p>

Choose Ollama as the provider, set the base URL to `http://localhost:11434`, and choose an Ollama model.

```shell
openclaw onboard
```

<p className="manual-step-title">3. Connect a channel</p>

```shell
openclaw channels add
```

<p className="manual-step-title">4. Start OpenClaw</p>

```shell
openclaw tui
```

To stop the gateway:

```shell
openclaw gateway stop
```
