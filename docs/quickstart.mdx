Follow this path to install Ollama, run a model, and use it from an app, API request, or integration.

## 1. Install Ollama

Ollama is available for macOS, Windows, and Linux.

<a
  href="https://ollama.com/download"
  target="_blank"
  className="inline-block px-6 py-2 bg-black rounded-full dark:bg-neutral-700 text-white font-normal border-none"
>
  Download Ollama
</a>

After installing, open a terminal and check that Ollama is available:

```shell
ollama --version
```

## 2. Run a model

Start with `gemma4` for a local first run:

```shell
ollama run gemma4
```

Ask a question in the prompt that opens:

```text
Write a short haiku about local AI.
```

To leave the chat, type `/bye`.

<Note>
If you want a larger model or do not want to wait for a local download, sign in and use a cloud model such as `kimi-k2.5:cloud` or `qwen3.5:cloud`.
</Note>

### API example

Use `/api/chat` to send a message to the model:

```shell
curl http://localhost:11434/api/chat -d '{
  "model": "gemma4",
  "messages": [
    {
      "role": "user",
      "content": "Why is the sky blue?"
    }
  ]
}'
```

### Integration examples

Use `ollama launch` when you want Ollama to configure a supported tool for you:

```shell
ollama launch claude
```

```shell
ollama launch openclaw
```

```shell
ollama launch codex-app
```

Claude Code is a good first choice for terminal-based coding, OpenClaw is a good first choice for a personal assistant, and Codex App is a good first choice for desktop coding and review workflows.
