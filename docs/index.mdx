Start building with open models.

Download Ollama, run a model, then use it from an app or the API.

## Quickstart

Install Ollama and run your first model.

<CardGroup cols={2}>
  <Card title="Download Ollama" icon="download" href="https://ollama.com/download">
    Install Ollama on macOS, Windows, or Linux.
  </Card>

  <Card title="Read the quickstart" icon="rocket" href="/quickstart">
    Go from install to your first response.
  </Card>
</CardGroup>

Run a local model from your terminal:

```shell
ollama run gemma4
```

Or sign in and run a cloud model:

```shell
ollama signin
ollama run kimi-k2.7-code:cloud
```

## Choose a model

Use a local model when you want it on your machine. Use a cloud model when you want a larger model without downloading it.
