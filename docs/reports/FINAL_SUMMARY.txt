HOW TO GET STARTED
================================================================================

STEP 1: Read Documentation
────────────────────────────────────────────────────────────────────────────
1. Start with: DELIVERABLES_INDEX.md
2. Read: COMPLETION_REPORT.md (5 min summary)
3. Reference: PROJECT_STATUS.md (current state)
4. Plan next: NEXT_ACTIONS.md (what to do next)

STEP 2: Set Up Development Environment
────────────────────────────────────────────────────────────────────────────
1. Follow: DEVELOPMENT_SETUP.md
2. Create virtual environment: python3 -m venv venv
3. Activate: source venv/bin/activate
4. Install: pip install -e .
5. Start Docker: docker-compose -f docker-compose.minimal.yml up -d

STEP 3: Run Quality Checks
────────────────────────────────────────────────────────────────────────────
1. Tests: pytest tests/ -v --cov=ollama
2. Types: mypy ollama/ --strict
3. Linting: ruff check ollama/
4. Format: black ollama/ tests/

STEP 4: Start Development Server
────────────────────────────────────────────────────────────────────────────
1. Activate venv: source venv/bin/activate
2. Start server: uvicorn ollama.main:app --reload --host 0.0.0.0 --port 8000
3. Test: curl http://YOUR_REAL_IP:8000/api/v1/health

STEP 5: Make Your First Commit
────────────────────────────────────────────────────────────────────────────
1. Make changes to code
2. Stage: git add .
3. Commit: git commit -m "type(scope): description"
   (Pre-commit hooks will run automatically)
4. Push: git push origin branch-name
