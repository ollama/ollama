# Personal AI Agent - Architecture & Documentation

## Overview

This is a **personal AI assistant** that runs locally on your Mac Mini. You message it via Telegram, and it can do almost anything: book appointments, search the web, send emails, manage your calendar, and more.


| Problem | Solution |
