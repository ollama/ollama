# 安卓TV安裝指南 (Android TV Installation Guide)

本指南將幫助您在安卓TV上安裝和配置TVBox直播功能。

## 📱 前置要求

- 安卓TV設備 (Android TV 5.0+)
- 網絡連接
- 可以安裝第三方應用的權限

## 🚀 安裝步驟

### 選項A: 使用 TVBox 應用

#### 步驟1: 下載並安裝 TVBox

1. 在安卓TV上打開瀏覽器或使用U盤
2. 下載 TVBox APK 文件
3. 允許從未知來源安裝應用
4. 安裝 TVBox 應用

#### 步驟2: 配置直播源

1. 打開 TVBox 應用
2. 點擊設定圖標 (通常在右上角)
3. 選擇「配置地址」或「配置管理」
4. 有兩種方式添加配置:

   **方法A: 通過URL (推薦)**
   - 如果配置文件已上傳到網絡，輸入 `config.json` 的完整URL
   - 例如: `https://your-domain.com/examples/tvbox/config.json`
   - 點擊確認

   **方法B: 本地導入**
   - 將 `config.json` 複製到TV設備
   - 在TVBox中選擇「本地導入」
   - 瀏覽並選擇配置文件

5. 配置加載完成後，返回主界面
6. 點擊「直播」標籤
7. 選擇您想觀看的頻道開始播放

### 選項B: 使用 TiviMate

#### 步驟1: 安裝 TiviMate

1. 從 Google Play Store 搜索並安裝 "TiviMate IPTV Player"
2. 或者從官方網站下載 APK 並手動安裝

#### 步驟2: 添加播放列表

1. 打開 TiviMate 應用
2. 點擊「添加播放列表」
3. 選擇「URL」
4. 輸入 M3U 文件的 URL 或使用本地文件:
   - 網絡URL: `https://your-domain.com/examples/tvbox/live.m3u`
   - 本地文件: 瀏覽並選擇 `live.m3u`
5. 設置播放列表名稱 (例如: "TVBox 直播")
6. 點擊「下一步」完成設置
7. 返回主界面，選擇頻道開始觀看

### 選項C: 使用網頁播放器

#### 步驟1: 部署網頁播放器

1. **使用 GitHub Pages (免費):**
   ```bash
   # 如果這是您的倉庫
   # 在 GitHub 倉庫設置中啟用 GitHub Pages
   # 選擇主分支和 /examples/tvbox 目錄
   ```

2. **使用本地服務器:**
   ```bash
   # 使用 Python 快速啟動本地服務器
   cd examples/tvbox
   python3 -m http.server 8000
   # 或使用 Node.js
   npx http-server -p 8000
   ```

3. **使用雲端託管:**
   - 上傳到任何靜態網站託管服務
   - 記錄播放器的 URL

#### 步驟2: 在TV瀏覽器中打開

1. 在安卓TV上打開瀏覽器 (如 Chrome、Firefox)
2. 輸入播放器的 URL:
   - GitHub Pages: `https://username.github.io/ollama/examples/tvbox/player.html`
   - 本地服務器: `http://192.168.x.x:8000/player.html`
3. 使用遙控器導航:
   - **上/下鍵**: 選擇頻道
   - **確認鍵**: 播放選中的頻道
   - **返回鍵**: 退出全螢幕
4. 點擊控制按鈕:
   - 🔄 重新加載
   - ⛶ 全螢幕

## 🎮 遙控器操作

### 通用操作
- **方向鍵**: 導航菜單
- **確認/OK**: 選擇/播放
- **返回**: 返回上一級/退出全螢幕
- **菜單**: 打開設置
- **播放/暫停**: 控制播放

### TVBox 專用
- **長按OK**: 顯示播放設置
- **音量鍵**: 調節音量
- **頻道鍵**: 切換上下頻道

### TiviMate 專用
- **長按確認**: 頻道信息
- **頻道+/-**: 快速切換頻道
- **快進/快退**: 時移功能 (如果支持)

## 🔧 故障排除

### 問題1: 無法播放某些頻道
**解決方案:**
- 嘗試其他頻道確認網絡連接
- 某些源可能暫時不可用
- 檢查防火牆設置
- 嘗試切換播放器類型 (系統播放器/IJK/Exo)

### 問題2: 播放卡頓
**解決方案:**
- 檢查網絡速度 (建議 5Mbps+)
- 關閉其他佔用網絡的應用
- 使用有線連接代替WiFi
- 降低視頻解析度

### 問題3: 配置無法加載
**解決方案:**
- 確認配置文件 URL 正確
- 檢查文件格式 (必須是有效的 JSON/M3U)
- 確保網絡可以訪問該 URL
- 嘗試使用本地導入方式

### 問題4: 應用閃退
**解決方案:**
- 清除應用緩存和數據
- 重新安裝應用
- 檢查安卓系統版本兼容性
- 釋放設備存儲空間

### 問題5: 遙控器無響應
**解決方案:**
- 檢查遙控器電池
- 重新配對遙控器
- 使用手機作為遙控器 (下載遙控器應用)
- 嘗試USB鼠標或鍵盤

## 📞 技術支持

4. 確保直播源仍然有效
