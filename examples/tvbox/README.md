# TVBox Live Streaming Sources

This directory contains M3U playlist files for IPTV/live streaming sources that can be used with TV Box applications.

## Files

- `live.m3u` - Collection of live streaming sources including:
  - 直播源倉庫 (Live Source Repository) - Various streaming sources from different providers
  - 單線直播 (Single Line Live) - Direct streaming links
  - 文字源 (Text Sources) - Text-based source links

## Usage

These M3U playlist files can be loaded into compatible TV Box applications or IPTV players that support the M3U format.

### Compatible Applications

- TiviMate
- Various Android TV Box applications
- IPTV players supporting M3U playlists
